# IMDb Sentiment Analysis using Sequence Models

This project implements and compares three recurrent neural network architectures for binary sentiment classification on the IMDb Movie Reviews dataset using PyTorch:

- Vanilla RNN
- LSTM
- GRU

The models are trained and evaluated under similar settings to compare their performance and identify the best-performing architecture.

## Dataset

- IMDb Movie Reviews Dataset
- Binary sentiment classification (Positive / Negative)

## Repository Contents

- `IMDb_Sentiment_Analysis.ipynb` – Complete implementation, training, evaluation, and comparison of all models.
- `best_rnn_model.pt` – Best-performing Vanilla RNN model.
- `best_lstm_model.pt` – Best-performing LSTM model.
- `best_gru_model.pt` – Best-performing GRU model.

## Technologies

- Python
- PyTorch
- NumPy
- Matplotlib

## Models Implemented

- Vanilla RNN
- LSTM
- GRU

## Objective

Compare the performance of recurrent neural network architectures for sentiment classification and determine the most effective model on the IMDb dataset.
